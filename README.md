# modern-react-cli
用于创建现代react模板的脚手架
## Getting started
```bash
//询问式创建模板，并自动下载依赖
npx modern-react-cli create myproject

cd myproject

//开启项目
npm|pnpm|yarn run dev
```
##commitlint
开启commitlint后，使用 npm|pnpm|yarn run cz代替 git commit可以获得友好的提示
